Este repositório contém arquivos necessarios para usar as rotas do projeto knowflow


## ⚙️ Tecnologias Utilizadas

- Node.js
- Express
- Jest (ou Mocha/Chai)
- Supertest (para testes de rotas HTTP)
- dotenv (se necessário)
